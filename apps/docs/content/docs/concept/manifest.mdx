# Продуктовый манифест: FOD

Этот документ — ваш «мастер-промпт» для всей разработки.

## Core Vision
FOD — это многопользовательский игровой клиент и сервер на базе WebGL, созданный для демонстрации высокопроизводительного геймплея прямо в браузере.

## UX Strategy: The Golden Path
1. **Onboarding**: Быстрый гостевой вход или авторизация.
2. **Game Lobby**: Выбор комнаты/сервера или создание новой игровой сессии.
3. **Gameplay**: Плавный рендеринг через WebGL с низкой задержкой ввода.
4. **Real-time Sync**: Быстрое сетевое взаимодействие через WebSockets с Elysia-сервером.

## UI & Visual Direction
- **Стиль**: Футуристический игровой UI с темной темой и высокой контрастностью.
- **Рендеринг**: Аппаратно ускоренный WebGL.
- **Типографика**: Современный sans-serif (Inter/Geist/Outfit).
