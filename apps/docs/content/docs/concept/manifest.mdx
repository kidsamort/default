# Продуктовый манифест: CV.AI

Этот документ — ваш «мастер-промпт» для всей разработки.

## 的核心 (Core Vision)
CV.AI — это интеллектуальный брокер между опытом кандидата и требованиями работодателя. Мы решаем «проблему чистого листа» через ИИ.

## UX Strategy: The Golden Path
1. **Onboarding**: Старый PDF -> JSON Master Profile.
2. **Smart Input**: Ссылка на вакансию или текст.
3. **Generation**: LLM берет вакансию + Master Profile -> JSON.
4. **Split-Screen Review**: Редактирование на лету с подсветкой ключевиков.
5. **O2O**: PDF с QR-кодом, ведущим на веб-версию.

## UI & Visual Direction
- **Стиль**: Строгий утилитарный минимализм (**Notion / Linear vibe**).
- **Сетка**: 8pt grid system.
- **Цвета**: Монохром.
- **Типографика**: Geist (sans-serif).
