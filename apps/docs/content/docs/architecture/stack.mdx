# Технический Стек

## Принципы
- **Bun**: Рантайм и менеджер пакетов.
- **ElysiaJS**: Высокопроизводительный Backend.
- **Next.js**: Frontend и Docs.
- **TypeBox**: Единая типизация через `@fod/schema`.

## Monorepo
- `apps/promo`: Next.js 16 лендинг.
- `apps/client`: Vite + React 19 WebGL игровой клиент.
- `apps/server`: Elysia API & WebSocket сервер.
- `apps/docs`: Fumadocs портал документации.
- `apps/storybook`: Storybook 10 для компонентов.
- `packages/schema`: TypeBox DTOs.
- `packages/db`: Drizzle ORM.
