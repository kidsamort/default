# Технический Стек

## Принципы
- **Bun**: Рантайм и менеджер пакетов.
- **ElysiaJS**: Высокопроизводительный Backend.
- **Next.js**: Frontend и Docs.
- **TypeBox**: Единая типизация через `@cvai/schema`.

## Monorepo
- `apps/web`: Next.js 16.
- `apps/api`: Elysia.
- `apps/docs`: Fumadocs.
- `packages/schema`: TypeBox DTOs.
- `packages/db`: Drizzle ORM.
