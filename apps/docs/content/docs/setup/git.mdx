# Настройка Git

Репозиторий инициализирован с использованием современных стандартов для монорепозиториев и AI-ассистентов.

## Правила игнорирования (.gitignore)

Мы используем детальный `.gitignore` для поддержания чистоты репозитория.

### Что игнорируется:
- `node_modules/` — зависимости.
- `.next/`, `.turbo/`, `dist/`, `build/` — артефакты сборки.
- `.env*.local` — локальные секреты.
- `.agents/`, `.gemini/` — временные данные и логи AI-ассистентов.

## Состояние репозитория

- **Ветка по умолчанию**: `master`.
- **Biome Integration**: Мы не коммитим `.biome/` кеш.
- **Turbo Integration**: Входные данные для Turbo включают `biome.json`.

## Рекомендации
- Не добавляйте в гит файлы с секретными ключами.
- Перед коммитом всегда запускайте `bun run lint`.
