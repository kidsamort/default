# Развертывание и Разработка

## Быстрый старт
```bash
# 1. Установка всех зависимостей во всех воркспейсах
bun install

# 2. Подготовка инфраструктуры (в корне проекта или папке docker)
docker-compose -f docker/docker-compose.yml up -d

# 3. Применение миграций (будет доступно после настройки @fod/db)
bun run db:push

# 4. Запуск всех приложений в dev-режиме
bun dev
```

## Конфигурация Docker
По умолчанию:
- **Postgres**: `localhost:5432` (User: `user`, Pass: `password`, DB: `fod`)
- **Redis**: `localhost:6379`
- **Frontend**: `localhost:3000`
- **API**: `localhost:3001`
- **Docs**: `localhost:3002` (или через порт Next.js в воркспейсе docs)

## Рабочие процессы (CI/CD)
Проект использует **Turborepo** для кеширования билдов.
- `turbo build`: Сборка всех пакетов.
- `turbo lint`: Проверка качества кода.
