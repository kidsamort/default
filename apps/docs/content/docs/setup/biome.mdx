# Настройка Biome

Мы используем **Biome** как единый инструмент для линтинга и форматирования во всем монорепозитории. Это заменяет связку ESLint + Prettier одним высокопроизводительным решением.

## Особенности
- **Скорость**: Biome работает в десятки раз быстрее, чем ESLint/Prettier.
- **Единообразие**: Один инструмент для всех задач (линтинг, форматирование, организация импортов).
- **Готовность к монорепозиторию**: Конфигурация в корне (`biome.json`) и интеграция с Turbo.

## Конфигурация

Корневой файл настроек: `biome.json`. Он распространяется на все пакеты и приложения:
- `apps/web`
- `apps/api`
- `apps/docs`
- `packages/*`

### Команды

Вы можете запускать эти команды из корня или внутри любого пакета:

```bash
# Проверка всего репозитория (через Turbo)
bun run lint

# Форматирование кода
bun run format

# Прямой запуск Biome для исправления ошибок
bunx biome check --write .
```
